# 失語症の調査結果（2026-04-14時点）

## 1. この調査の目的
「特定の単語が打てない」という課題を、失語症の特性に照らして整理し、練習ソフト要件に反映できる知見をまとめる。

## 2. 要点サマリ
- 失語症は「言語」の障害で、発話だけでなく、聞く・読む・書くにも影響しうる。
- 主因は脳卒中（特に左半球関連）だが、頭部外傷・腫瘍・感染・進行性疾患でも起こる。
- 失語症は知能低下そのものではない（思考は保たれているが、言語化が難しい状態がある）。
- リハビリでは言語聴覚療法（SLT）が中心で、機能改善に有効という報告がある。
- 日本では、失語症者向け意思疎通支援（自治体事業）が制度化されている。

## 3. 医学的整理
### 3.1 定義と症状
- NIDCDは失語症を「言語を司る脳領域の損傷で生じる障害」とし、表出・理解に加えて読み書きへの影響を説明している。
- 国立障害者リハビリテーションセンター病院でも、失語症は「聞く・話す・読む・書く・計算」の障害として整理されている。

### 3.2 原因
- NIDCDでは、失語症の主因を脳卒中としている。
- そのほか、外傷性脳損傷、脳腫瘍、脳感染などでも生じうる。

### 3.3 誤解されやすい点（重要）
- 厚生労働省の説明では、失語症は言語中枢損傷による障害で「物事を考える機能は保たれている」が、言語処理が困難になるとされる。
- American Stroke Associationも、失語症は知能を損なうものではない旨を明示している。

### 3.4 評価・介入
- NIDCD/ASHAとも、評価と介入の中心は言語聴覚士（SLP/ST）による包括的評価と訓練。
- Cochraneレビュー（2016）は、脳卒中後失語症に対するSLTが、無治療比較で機能的コミュニケーション等の改善に有効と報告。
- ただし同レビューの証拠対象期間は2015年までで、最新研究の継続確認は今後必要。

## 4. 日本の制度・社会資源
- 厚労省「意思疎通支援」では、失語症者向け支援（会話理解・表現補助、絵や道具の利用等）が明記されている。
- 実施主体は自治体（市区町村・都道府県）で、派遣・養成・広域調整の仕組みがある。

## 5. 本プロダクト（練習ソフト）への示唆
- 「発音できるか」を前提にしないUIが必須。
- 1つの単語に対し、かな/カナ/漢字/英字など複数表記を同時に提示する。
- 正答/誤答だけでなく、「どこで詰まったか（想起・選択・タイピング）」を分離記録する。
- 苦手単語の反復は、短時間・高頻度に寄せつつ離脱率を監視する（高強度訓練の中断リスクを考慮）。
- 支援者・家族が伴走できるモード（ヒント提示、絵カード、会話補助）を将来拡張として持つ。

## 6. この調査の限界
- 本調査は公開情報に基づく整理で、診断・治療方針を示すものではない。
- Cochrane根拠は重要だが、公開要約は2016年版のため、最新メタ解析の継続レビューが望ましい。
