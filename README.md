# word_practice

特定の単語が打てない課題を克服するための練習ソフトのプロジェクトです。

## 現在のフェーズ
- 音声機能（録音・音声認識・音声解析）は未実装
- まずはドキュメント設計と練習要件の整理を進める

## ドキュメント
- `doc/課題.md`: 課題メモ
- `doc/要件定義.md`: 要件定義（練習ソフト前提）
- `doc/失語症調査.md`: 失語症の調査結果
- `doc/コミットルール.md`: このプロジェクトの日本語コミットルール

## 方針
- `word_practice` をリポジトリルートとして運用
- `doc/` 配下を主要成果物として更新
- コミットメッセージは日本語で記述
