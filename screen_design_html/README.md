# screen_design_html

画面設計用の静的HTMLモックです。

## 画面
- `index.html`: 練習 + 結果（メイン画面）
- `practice.html`: 互換リンク（indexへリダイレクト）
- `result.html`: 互換リンク（indexへリダイレクト）
- `settings.html`: 設定

## 観点
- 入力: ローマ字
- 表示: 漢字主表記
- スタート押下まで問題非表示
- 入力文字の直接表示（テキストボックスなし）
- リアルタイム判定（判定ボタンなし）
- 画面2分割（左: 練習/結果、右: 苦手傾向7件）
- 職業選択に応じた問題の自動登録
- LLMによる問題生成
